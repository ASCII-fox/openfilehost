Policies

This server's policies are:

File select

No file selected.

Important

The only time you will be shown your download key is after uploading. Without the download key, you cannot:

  1. Download the file.
  2. Query the file.
  3. Freely delete the file from the server. It will be stored until it expires.
Also understand that anyone you share the download key with will be able to do the above three actions.
